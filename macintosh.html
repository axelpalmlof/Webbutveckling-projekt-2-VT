Macintosh

Revolutionary user interface.

The graphical user interface used by apple computer's macintosh family of personal computers, is based on graphical representations of familiar office objects positioned on a two dimensional desktop workspace. Programs and data files are represented on screen by small pictures. An object is selected by moving a mouse over the real desktop which correspondingly moves the pointer on screen.
